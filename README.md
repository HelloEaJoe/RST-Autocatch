# RST-Autocatch 🤖

本项目由 NEU-Ocean Heart Group 开发，旨在实现自主水下机器人（AUV）的自主识别与抓球任务，面向 RST竞赛。

---

📌 项目简介

RST-Autocatch 是一个专为 AUV 设计的视觉识别与抓球系统。通过高效的图像处理算法与运动控制策略，系统能够在水下环境中自主识别目标球体，并完成精准抓取动作。

---

🚀 主要功能

· 目标识别：基于计算机视觉的水下球体识别

· 自主导航：AUV 的路径规划与运动控制

· 抓取机制：机械臂或抓取装置的自动控制

· 实时处理：支持实时图像处理与决策
