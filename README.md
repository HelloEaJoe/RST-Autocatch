## 📌 项目简介

RST-Autocatch 是一个专为 AUV 设计的视觉识别与抓球系统，由 **NEU-Aquacore** 开发，面向 RST 竞赛任务。

**当前版本说明**：本项目为 **初代原型代码**，聚焦于验证“水下自主识别 + 抓取控制”这一完整任务链路的可行性。通过高效的图像处理算法与基础运动控制策略，系统能够在水下环境中完成目标球体的识别与抓取动作，为后续功能迭代与性能优化奠定基础。

---

## 🚀 主要功能

| 功能模块 | 说明 |
| :--- | :--- |
| 🎯 目标识别 | 基于计算机视觉的水下球体识别（初代算法实现） |
| 🧭 自主导航 | AUV 的路径规划与基础运动控制 |
| 🦾 抓取机制 | 机械臂或抓取装置的自动控制逻辑 |
| ⚡ 实时处理 | 支持实时图像采集、处理与简单决策 |

> ⚠️ **注意**：本版本为初代原型代码，部分模块尚处于实验阶段，可能存在不稳定情况，欢迎反馈与改进。
